# SpeakLab

Speech coaching studio. Record yourself, get instant transcription via Groq Whisper, and AI coaching feedback via Gemini.

## Setup

1. Open `index.html` in Chrome — or host on GitHub Pages
2. Click ⚙ Settings → paste your API keys
3. Hit Record → speak → Stop → Analyze

## API Keys (both free)

- **Groq** (transcription): [console.groq.com/keys](https://console.groq.com/keys)
- **Gemini** (coaching): [aistudio.google.com/apikey](https://aistudio.google.com/apikey)

## Features

- Video recording with live camera feed
- Transcription with filler word highlighting
- AI speech coach with conversation memory
- Session library with transcript and chat history
- Videos save to a folder you choose (File System Access API)
- All data stays local — nothing stored on any server

## Stack

Single HTML file. No dependencies, no build step, no backend.
